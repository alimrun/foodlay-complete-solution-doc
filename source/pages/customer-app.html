Overview

Flutter application connecting to Laravel backend
Supports authentication, product browsing, and ordering
Targets Android and iOS platforms

Requirements

Flutter 3+
Dart 3+
Android Studio or Xcode
Devices or emulators

Setup

Get dependencies

flutter pub get

Generate code

flutter pub run build_runner build --delete-conflicting-outputs

Environment

Config file

API_BASE_URL=
API_KEY=
SENTRY_DSN=
FLAVOR=dev|staging|prod

Flavors

lib/
  main_dev.dart
  main_staging.dart
  main_prod.dart

Project Structure

lib/core: configuration and utilities
lib/data: models and data sources
lib/features: UI screens and state management
lib/widgets: shared components

Run and Debug

flutter run --flavor dev -t lib/main_dev.dart
flutter run --flavor staging -t lib/main_staging.dart
flutter run --flavor prod -t lib/main_prod.dart

Build and Release

Android

flutter build apk --flavor prod -t lib/main_prod.dart

iOS

flutter build ipa --flavor prod -t lib/main_prod.dart

API Usage

Use token-based authentication
Handle pagination for lists
Use secure storage for sensitive data

Testing

flutter test

Troubleshooting

Check device logs and network requests
Verify API base URL and flavor configuration
Clear caches and reinstall app if needed
