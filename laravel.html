Overview

Laravel backend powering two Flutter applications
RESTful APIs for authentication, profile, products, orders
Queue workers for emails, notifications, and background jobs

Requirements

PHP 8.2+
Composer 2+
MySQL 8 or MariaDB 10+
Redis for caching and queues
Node.js for asset tooling if applicable

Setup

Install dependencies

composer install

Environment

cp .env.example .env
php artisan key:generate

Database

php artisan migrate --seed

Core variables

Project Structure

app: application code and domain logic
routes: API and web routes
database: migrations, seeders, factories
config: application configuration
resources: views and assets

Database

Migrations

php artisan migrate

Seeders

php artisan db:seed

API

Authentication and token issuing
CRUD endpoints for core resources
Pagination and filtering for collections

Run server

php artisan serve

Queues and Jobs

php artisan queue:work

Broadcasting

Driver options: log, pusher, redis
Use channels for real-time updates

Testing

php artisan test

Deployment

Optimize configuration and routes
Run migrations and seeders
Start queue workers and horizon if used
php artisan config:cache
php artisan route:cache
php artisan migrate --force

Troubleshooting

Check logs in storage/logs
Verify environment variables and cache
Ensure queue workers are running
