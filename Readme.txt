1. If you are newly installing the admin panel, please only use the "Admin Panel New Install V1.1" zip file.
   You do not need to use the Admin Panel Update zip file.
   The Admin Panel Update zip file is only for updating from V1.0 to V1.1.

2. To update the admin panel from V1.0 to V1.1, log in to your Admin Panel,
   navigate to yourdomain.com/updater, upload the "Admin Panel Update to V1.1.zip" file,
   and click Update. Must follow the documentation.

3. As the Customer App and Deliveryman App are developed using Flutter,
   use the "Customer App" and "Deliveryman App" folders for full source code builds.
   Must follow the documentation.

4. If you need only the changed files for mobile apps in V1.1,
   check the "Customer App Changes from V1.0 to V1.1" and
   "Deliveryman App Changes from V1.0 to V1.1" folders.
   Otherwise, please use the full "Customer App" and "Deliveryman App" folders.

Documentation - https://dayonesoft.com/foodlay/documentation/
